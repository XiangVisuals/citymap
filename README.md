# 3D City Terrain Sandbox 
**3D 城市地形沙盘编辑器**

一个轻量级、高性能且极具设计感的 Web 端 3D 城市生成与沙盘编辑器。只需导入标准的地图和高程数据，并支持丰富的二次编辑与项目级打包导出。

---

## ✨ 核心特性

### 🎨 极致美学 (Aesthetic)
* **纸模型质感**：抛弃繁杂的贴图，采用高明度、低粗糙度的全局白模体系。
* **莫兰迪配色**：系统内置深度调优的色彩基调（极浅灰路网、豆沙绿公园、静谧蓝海湾）。
* **无缝暗黑模式**：一键切换昼夜主题，建筑高亮与环境光影自动流转。
* **微交互运镜**：3D 模式下选中任意建筑，自动触发平滑的“无人机环绕航拍”视角。

### 🗺️ 硬核 GIS 数据融合
* **OSM 智能解析**：自动读取 `.osm` / `.xml` 文件，智能识别并分类生成建筑 (Building)、路网 (Highway)、水系 (Water)、绿地 (Park)。
* **DEM 高程贴合**：支持导入 `.tif` (GeoTIFF) 高程数据，所有建筑与路网自动沿真实地形起伏。
* **超大城市框选**：内置 2D Canvas 框选预览工具，只渲染所选区域，极大优化大规模城市渲染性能。
* **路网分级引擎**：支持 1~6 级道路过滤（高速路至社区小径），可按需预览并精准导出。

### 🏗️ 高自由度对象编辑
* **空间微调**：支持对建筑进行精细的 X/Z 轴坐标偏移、高度调整及自转角度修改。
* **GLB 高精模型替换**：支持上传 `.glb` / `.gltf` 模型替换简易块状建筑。引擎会自动剥离原模型材质，强制赋予统一的系统光影白模质感。
* **自定义绘制**：支持在任意空地单击快速放置新的自定义块状建筑。

### 📦 闭环的项目流转 (Import & Export)
* **ZIP 工程化打包**：一键将所有手动微调、增删的建筑及外部高精模型（GLB）打包为 `.zip` 项目文件。
* **纯正 OSM 拓扑还原**：基于逆墨卡托投影算法，将 3D 坐标无损逆向转化为标准的经纬度 `<node>` 和 `<way>`，并使用 `<tag>` 自动关联本地 3D 模型体系。
* **无缝二次复原**：直接将导出的 `.zip` 拖入系统，即可 100% 还原上一次的沙盘编辑现场。

## 🛠 技术栈 (Tech Stack)

* **核心渲染**：`Three.js` (WebGL) + `EffectComposer` (FXAA抗锯齿)
* **UI 框架**：`TailwindCSS` (响应式与暗色模式支持)
* **地理数据处理**：`GeoTIFF.js` (高程解析)
* **文件打包**：`JSZip` (前端文件压缩与解压)
* **架构**：零构建构建依赖，纯前端原生 ES Module 模块化运行。

## 🚀 快速上手 (How to Use)

1. **获取数据**：
   * 前往 [OpenStreetMap 导出所需城市的 `.osm` 地图数据](https://www.openstreetmap.org/export)。
   * （可选）获取对应区域的 GeoTIFF 高程数据。
2. **初始化沙盘**：
   * 打开 `index.html`，点击“选择 OSM 文件”导入基础路网。
   * 如果地图过大，可以在弹出的预览框中画框选取核心区域。
3. **注入高程**（可选）：
   * 在左侧控制面板点击“上传 DEM 地形”，建筑会自动高低起伏。
4. **编辑与替换**：
   * 点击选中任意建筑，在右侧面板修改高度/位置，或上传高质量的 GLB 替换单体建筑。
5. **打包导出**：
   * 点击左下角“打包导出 OSM + 模型 (.zip)”保存您的沙盘项目。

## 📌 数据识别规范说明

系统会依据以下 OSM 标签自动分类网格：
* **建筑 (`Building`)**：包含 `building=*` 的闭合 Way。
* **路网 (`Highway`)**：包含 `highway=*` 的节点，按 motorway(1) 到 residential(5) 自动分级与分配线宽。
* **公园 (`Park`)**：包含 `leisure=park/garden` 或 `landuse=grass/forest` 且闭合的面。
* **水系 (`Water`)**：包含 `natural=water/bay` 或 `waterway=riverbank/dock` 且闭合的面。
